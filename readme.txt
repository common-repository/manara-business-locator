== Description ==


Manara Business Locator depends on Google Maps, so always you will have an updated system and You have a tool to add a list of business on the map, it is very simple and all the hard work is already done for you, the only thing you will need to do is create a list of companies that you will need to add on the map.
All companies added on the list will be display and Manara Business Locator will calculate how many miles away you are from the post code you have input on the field.
The company that is nearby you will be added on the map and the details of the company will be displayed underneath the Map.  


== Installation ==

1. Visit 'Plugins > Add New'
2. Search for 'Manara Business Locator'
3. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
4. Activate the plugin through the 'Plugins' screen in WordPress
5. Optional but if you know that Search will past 2500 searchs per day you need to add a App Key from google.

== Frequently Asked Questions ==
= The plugin will modify the content in the database =
Yes, it will create a new table and all data will saved there but it will not interfere with wordpress core settings.  Observation if you uninstall the system also you drop the table and all data will be lost, make sure you have a back of your data base before you remove the plugin.
